# Spring Boot Beans and Dependency Injection Demo

This project demonstrates core Spring Boot concepts:

## Features

- Bean Creation using @Bean
- @Configuration
- Dependency Injection
- @Qualifier
- Bean Lifecycle
  - @PostConstruct
  - @PreDestroy
- Interface-based Design

## Technologies Used

- Java 17
- Spring Boot
- Maven

## Concepts Practiced

- Spring IoC Container
- Bean Management
- Constructor Injection
- Multiple Bean Handling
- Qualifier Annotation
- Bean Lifecycle Management
